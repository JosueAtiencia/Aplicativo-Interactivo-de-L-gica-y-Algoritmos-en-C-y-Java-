# Aplicativo-Interactivo-de-L-gica-y-Algoritmos-en-C-y-Java-

La presente práctica tiene como finalidad desarrollar un aplicativo interactivo en Java 
que permita aplicar los conocimientos adquiridos en la asignatura de Algoritmos y Lógica de 
Programación. El programa implementa estructuras fundamentales como variables, 
condicionales, ciclos, arreglos y manejo de archivos, permitiendo realizar operaciones 
matemáticas, registrar estudiantes con sus respectivas notas y almacenar los resultados de 
manera permanente en un archivo de texto. 

Además, este proyecto busca fortalecer el uso de validaciones, procesamiento de datos 
y persistencia de información, integrando también herramientas de control de versiones 
mediante GitHub para una mejor organización y administración del código fuente. Con el 
desarrollo de esta práctica se refuerzan habilidades de programación lógica y estructurada 
aplicadas a problemas reales.
